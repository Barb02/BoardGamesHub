# Project abstract
### Board Games Hub
Web page with the intention of comparing board games prices across multiple web stores, providing their current price and showcase their historic lowest prices.

Software used:
* Frontend - Reactjs
* API - SpringBoot
* Backend - Postgresql
* Repository - Github 
* Backlog - Github
* Agile Board - Github
